Constellation Planetarium — prototype

Click a name to open. Default view is the canonical "destiny shape" you see on the cert. The constellation auto-rotates slowly — drag to take over, release and it picks up where you let off. The 2D pattern is the projection along the Z axis; rotating uncloaks the depth scatter.

notes
· Planetarium runs through CosmicPlanetarium.open() — same module the cert pages will use.
· Each star's Z depth is derived from its own per-star seed (mocked here from heart's content_hash + index).
· Default view = canonical 2D pattern. Auto-rotate ~120s per revolution.
· Heart = α (spectral-tinted). Current star highlighted with radio pulse in rarity-tier color.
